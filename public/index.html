Welcome to BetaDoc API

This is the backend API for the BetaDoc healthcare platform. The API provides endpoints for patient and doctor management, consultation bookings, and more.

API Endpoints

The API provides the following main endpoints:

  • /api/auth - Authentication endpoints for patients and doctors
  • /api/users - User management endpoints
  • /api/consultations - Consultation booking and management
